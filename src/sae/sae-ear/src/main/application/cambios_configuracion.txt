Para la integración con CAS:

1 - Añadir el certificado digital del servidor CAS al truststore usado por el servidor Wildfly
2 - Configurar el Wildfly
    2.1 - Actualizar el módulo de login (jar y xml)
    2.2 - Añadir el login module SAEAutoLoginModule al security domain
        <login-module code="uy.gub.imm.sae.login.SAEAutoLoginModule" flag="sufficient" module="lat.sofis.saelogin">
            <module-option name="dsJndiName" value="java:/postgres-sae-ds"/>
            <module-option name="enabled" value="true"/>
        </login-module>
    2.3 - Añadir cache-type="default al security domain
    2.4 - Cambiar spec-descriptor-property-replacement a "true"        
    2.5 - Añadir las system-properties
	<property name="sae.server.name" value="localhost:8443"/>
        <property name="cas.server.url" value="https://guri-test.sofis.com.uy:8443/cas"/>
